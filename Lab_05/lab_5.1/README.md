# Lab 5.1: Children in React Components

## Обзор
Данная лабораторная работа посвящена изучению концепции `children` в React. Реализованы компоненты, демонстрирующие использование children для создания гибких и переиспользуемых компонентов.

## Цель работы
Научиться работать с `children` в React компонентах, понимать как передавать и렌дерить содержимое внутри компонентов, создавать переиспользуемые компоненты-контейнеры.

## Структура проекта в src:
*   `UserCard.tsx` — Компонент карточки пользователя. Принимает пропсы `user`, `isActive` и `children`. Демонстрирует использование `children` для вывода дополнительного содержимого.
*   `SkillList.tsx` — Компонент списка навыков. Выводит массив навыков с цветовой кодировкой уровня (Advanced - зелeный, Intermediate - оранжевый, Beginner - красный).
*   `App.tsx` — Корневой компонент, демонстрирующий использование `UserCard` с `children`.
*   `types.tsx` — Файл с TypeScript типами: `User` и `Skill`.

## Реализованный функционал
1.  **Children Concept:** Компонент `UserCard` принимает `children` и рендерит их внутри компонента.
2.  **Props Usage:** Компоненты получают данные через пропсы и конфигурируют поведение (например, `isActive` изменяет фон карточки).
3.  **Conditional Styling:** Использование условного стилирования на основе пропсов (`isActive` влияет на цвет фона).
4.  **List Rendering:** Компонент `SkillList` демонстрирует использование `.map()` для вывода массива элементов.
5.  **TypeScript Integration:** Использование интерфейсов для типизации пропсов.

## Методология
Основной акцент сделан на гибкость компонентов — `UserCard` может содержать любой контент через механизм `children`. Это позволяет использовать один компонент для разных целей.
