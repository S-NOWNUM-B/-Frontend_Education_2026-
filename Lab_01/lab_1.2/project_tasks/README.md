# Lab 1.2: React Project Setup

**Node Version:** 25.2.1
**pnpm Version:** 10.28.1

## Команды для запуска преокта

```bash
# 1) Войти в папку проекта
cd project_tasks

# 2) Установка всех зависимостей
pnpm install

# 3) Запуск преокта в редиме разработчика
pnpm run dev
```

## Используемые технологии
Проект создан с помощью Vite
