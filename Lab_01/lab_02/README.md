# Lab 1.2: React Project Setup & Minimal App

## Обзор
Лабораторная работа посвящена развертыванию современного React-проекта и реализации базового интерактивного компонента. Проект демонстрирует работу React как "View Layer" (слоя представления).

## Цель работы
Освоить инструментарий сборки (Vite), структуру React-проекта и механизмы управления состоянием и рендерингом.

## Технический стек и компоненты:
*   **Сборщик:** Vite + pnpm.
*   **App.jsx:** Корневой компонент, объединяющий логику переключателя (из Lab 1.1) и счетчика.
*   **Counter (в App.jsx):** Компонент-счетчик с кнопками Increment и Decrement, работающий через хук `useState`.
*   **Virtual DOM:** В проекте реализован принцип минимизации обновлений реального DOM через механизм Diffing & Patching.

## Методология и запуск
Приложение построено на базе функциональных компонентов. Логика обновлений отделена от разметки, что позволяет React эффективно перерисовывать только изменившиеся части интерфейса.

**Инструкция по запуску:**
1. Установите Node.js (версия 18+).
2. Выполните команду установки зависимостей:
   ```bash
   pnpm install

   pnpm run dev
   ```
