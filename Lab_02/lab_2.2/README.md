# Lab 2.2: Custom JSX Elements & Dynamic Props

## Обзор
В рамках этой работы изучены механизмы создания переиспользуемых компонентов и передачи данных через `props`. Внимание уделено динамическим атрибутам и работе со специальным свойством `children`.

## Цель работы
Научиться создавать кастомные JSX-элементы, которые принимают динамические данные, и использовать фрагменты для сохранения чистоты DOM-структуры.

## Компоненты лабораторной работы:
*   `Card.jsx` — Переиспользуемый компонент карточки. Принимает `title`, `children` и опциональный `className`. Демонстрирует использование выражений в атрибутах.
*   `ProductList.jsx` — Компонент для отображения каталога продуктов. Выполняет итерацию по массиву объектов и рендерит карточки продуктов, используя `id` в качестве ключа.
*   `Section.jsx` — Структурный компонент, возвращающий фрагмент с заголовком и вложенным контентом (`children`). Используется для логического разделения интерфейса без добавления лишних `div`-оберток.

## Технические требования
*   Использование только функциональных компонентов (React 18+).
*   Соблюдение camelCase в именовании пропсов и переменных.
*   Использование `className` для стилизации.
*   Отсутствие конкатенации строк вне JSX-выражений (использование шаблонных строк или выражений внутри `{}`).

## Методология
Подход основан на композиции компонентов: маленькие атомарные части (`Card`) объединяются в списки (`ProductList`) и секции (`Section`), которые затем собираются в конечное приложение в `App.jsx`. Это обеспечивает высокую читаемость и масштабируемость кода.
