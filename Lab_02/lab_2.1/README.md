# Lab 2.1: Fragment Drill & List Mapping

## Обзор
Данная лабораторная работа посвящена изучению основ рендеринга в React с использованием JSX-фрагментов и выражений. Основная цель — научиться группировать элементы без создания лишних узлов в DOM (избегая "div soup") и эффективно отображать массивы данных.

## Цель работы
Освоить синтаксис `React.Fragment` (`<>...</>`) и использование JavaScript-выражений внутри JSX для динамического вывода контента.

## Структура проекта в src:
Согласно требованиям, логика разделена на следующие компоненты:
*   `FragmentLayout.jsx` — Реализация **Task 1**. Компонент возвращает три соседних элемента (`header`, `main`, `footer`) внутри фрагмента, что исключает появление лишних оберток в HTML-дереве.
*   `ItemList.jsx` — Реализация **Task 2**. Компонент принимает массив данных и использует метод `.map()` для генерации списка `<li>` с обязательным использованием атрибута `key`.
*   `CombinedList.jsx` — Реализация **Task 3**. Комбинированный компонент, объединяющий заголовок, список и итоговую информацию (количество элементов) через JSX-выражения.
*   `App.jsx` — Корневой компонент, в котором происходит композиция вышеуказанных элементов.

## Реализованные задачи
1.  **Fragment-Only Layout:** Создана семантическая разметка без дополнительных `<div>`.
2.  **List Mapping:** Реализован вывод динамических списков с использованием уникальных ключей (`id`).
3.  **JSX Expressions:** Использованы возможности JavaScript для вычисления длины массива и вывода текстовых данных прямо в разметке.

## Инструкция по запуску
1. Установите зависимости: `pnpm install`
2. Запустите сервер разработки: `pnpm dev`
